# OpenCV Image Editor

An interactive image editor built using OpenCV.

## Features
- Image filters (grayscale, blur, edge detection)
- Morphological operations (dilation, erosion, opening, closing)
- Drawing tools (line, rectangle, circle, polygon)
- Brush & eraser
- Undo/Redo functionality
- Save output images

## Controls
- g: Grayscale
- b: Blur
- e: Edge detection
- t: Threshold
- r: Rotate
- m/n/j/k: Morphology
- R/G/B: Colors
- Mouse: Draw
- x: Eraser
- u/y: Undo/Redo
- s: Save
- q: Quit

## Tools
- Python
- OpenCV
- NumPy
